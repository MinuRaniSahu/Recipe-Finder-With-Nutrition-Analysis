# Recipe Finder with Nutrition Analysis

A Python-based desktop application built using Tkinter that allows users to:

- Search recipes using Spoonacular API
- Apply dietary filters
- View recipe details
- Analyze nutritional information
- Visualize data using Matplotlib

## Tech Stack
- Python
- Tkinter
- Spoonacular API
- Pandas
- Matplotlib

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python src/main1.py
   python src/main2.py
   python src/main3.py
   python src/main4.py
